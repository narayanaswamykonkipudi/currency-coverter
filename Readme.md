# 💱 Currency Converter App

A simple and interactive **Currency Converter Web App** built using **Python** and **Streamlit**.
It allows users to convert currencies in real-time using live exchange rates.

---

## 🚀 Features

* 🌍 Convert between multiple currencies
* ⚡ Real-time exchange rates (API integration)
* 📊 View all exchange rates
* 📜 Conversion history tracking
* 🧹 Clear history option
* 🎯 Simple and clean UI

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Requests (API handling)

---

## 📦 Installation

1. Clone the repository:

```
git clone https://github.com/narayanaswamykonkipudi/currency-coverter.git
```



3. Run the app:

```
streamlit run app.py
```

---

## ▶️ How to Use

1. Select **From Currency**
2. Select **To Currency**
3. Enter the **Amount**
4. Click **Convert**
5. View:

   * ✅ Converted value
   * 📊 Exchange rate
6. (Optional)

   * Enable **“Show all exchange rates”** to view full table
   * Check **Conversion History** below
   * Click **Clear History** to reset
